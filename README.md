# KeySensu

[Keybase](https://keybase.io) [Sensu](https://sensu.tips) Bot Client.

# How To use

1. Install Keybase.
2. pull KeySensu.
3. Register Sensu.
4. Settings `config.json`.
5. run `npm i`.
6. run `node realtime.js`.

## Streaming Mode

```
node realtime.js
```

## Rest Mode
```
node rest.js
```
